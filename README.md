# Cosmic Calendar

Visualize the history of the universe compressed into a single calendar year. This interactive 3D application takes you on a journey from the Big Bang to the present day.

## Features

- **Interactive Timeline**: Scrub through 13.8 billion years of history.
- **Cosmic Eras**: Visual color-coded zones representing Primordial, Stellar, Planetary, and Biotic eras.
- **3D Visualization**: Features a 3D starfield and spaceship overlay.
- **Responsive Design**: Works on desktop and mobile devices.

## Tech Stack

- Vue 3
- TypeScript
- Vite
- Three.js
- TailwindCSS

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Build for production:
   ```bash
   pnpm build
   ```
